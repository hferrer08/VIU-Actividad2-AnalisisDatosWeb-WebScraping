# Actividad 2 - Análisis de Datos Web

## Descripción

Proyecto académico correspondiente a la Actividad 2 de la asignatura Análisis de Datos Web.

El objetivo de esta actividad es realizar la extracción, procesamiento y análisis de datos climáticos obtenidos mediante técnicas de **Web Scraping**, utilizando herramientas del ecosistema Python como **Requests**, **BeautifulSoup** y **Pandas**.

## Tecnologías utilizadas

- Python 3
- Requests
- BeautifulSoup4
- Pandas
- Google Colab

## Objetivos

- Obtener información desde una página web mediante Web Scraping.
- Extraer datos tabulares de forma automatizada.
- Procesar y limpiar la información obtenida.
- Analizar los datos utilizando estructuras de datos de Pandas.
- Responder preguntas teóricas y prácticas relacionadas con la extracción de datos web.

## Fuente de datos

Sitio web utilizado para la extracción de información meteorológica:

<https://www.tutiempo.net/clima/ws-82210.html>
